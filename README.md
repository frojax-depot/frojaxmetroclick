# FROJAX FREE CLICK

Single-file metronome preview for GitHub Pages.

## How to deploy
1. Upload this folder to a GitHub repo.
2. Go to Settings → Pages → Deploy from a branch → main → /root.
3. Open the URL GitHub gives you.

## Notes
- Audio starts only after tapping **PLAY** (browser policy).
- Logos are in `/assets` and referenced by relative paths.
